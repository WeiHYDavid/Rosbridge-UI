# Rosbridge-UI

![](https://github.com/WeiHYDavid/Rosbridge-UI/blob/main/images/ui.png)



## 简介

用于建立ros话题消息与前后端之间通信的桥梁，该程序是对ros自带的rosbridge包上修改而来，添加了可视化交互界面以及对作为客户端的支持。



## 使用方式

打开前先运行roscore。
